# CE_COORDINATE validation checklist

- Pick an XYZ point while a non-world UCS is current; confirm the reported values are WCS/drawing coordinates.
- Place an XYZ MLeader and confirm its arrow points to the selected location.
- Batch-label at least ten Civil 3D COGO points.
- Confirm point name/number, raw description, easting, northing and elevation are correct.
- Create a coordinate cross and confirm its size is legible at the current drawing text height.
- Build a setting-out table from a mixture of AutoCAD DBPoints and Civil 3D COGO points.
- Confirm unsupported selected objects are reported as skipped.
- Confirm one UNDO reverses each command run.
